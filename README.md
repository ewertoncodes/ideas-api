
Clone o projeto:

```bash 
git clone https://github.com/ewertonorg/ideas-api.git 
```
Acesse o diretorio:

```bash 
cd ideas-api
```

Instale as dependências:

```bash
bundle install
```

```bash
rails db:setup
```

Rode os testes:

```bash
rspec spec
```

 Levante sua aplicação com o comando:

```bash
rails s
```
